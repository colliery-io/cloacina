# Start Here

Cloacina is one workflow orchestration engine you can run **two ways**: embed it
as a library inside your application, or operate it as a service. This section
orients you before you pick a path — what Cloacina is, whether it fits, and the
core vocabulary. When you're ready, choose a door:

- **[Embed the library]({{< ref "/embed" >}})** — integrate the engine into your own Rust or Python application.
- **[Run the service]({{< ref "/service" >}})** — operate Cloacina as a multi-tenant server with an API and UI.
