# Computation Graph Tutorials

Tutorials for Cloacina's **computation graph** system — the reactive scheduler that wires accumulators to reactors for event-driven, continuously executing computation graphs.

## Library (Embedded)

Start here. These tutorials teach you to define computation graphs, wire accumulators and reactors, and push events through the pipeline — all directly in your Rust application.
