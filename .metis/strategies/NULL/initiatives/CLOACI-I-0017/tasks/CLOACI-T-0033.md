# Split validator.rs into modular validators

## Objective

Split `src/registry/loader/validator.rs` (950 lines) into modular validators.

## Current State

Single validator doing package, task, workflow, and dependency validation.

## Target Structure

```
src/registry/loader/validators/
  mod.rs                    (~100 lines - public API)
  package_validator.rs      (~250 lines - package validation)
  task_validator.rs         (~250 lines - task validation)
  workflow_validator.rs     (~200 lines - workflow validation)
  dependency_validator.rs   (~150 lines - dependency validation)
```

## Acceptance Criteria

- [ ] Create `src/registry/loader/validators/` directory
- [ ] Split validators by concern
- [ ] Update `mod.rs` with re-exports
- [ ] All existing tests pass
- [ ] `cargo check` passes
