# Split reconciler.rs into modules

## Objective

Split `src/registry/reconciler.rs` (1,036 lines) into focused modules.

## Current State

The file mixes conflict detection, validation, and update scheduling.

## Target Structure

```
src/registry/reconciler/
  mod.rs           (~200 lines - Reconciler struct, public API)
  conflict.rs      (~300 lines - conflict detection logic)
  validation.rs    (~250 lines - reconciliation validation)
  scheduling.rs    (~250 lines - update scheduling)
```
