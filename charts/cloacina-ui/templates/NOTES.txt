IMPORTANT: the browser calls cloacina-server cross-origin. The server must
allow this UI's origin — set CLOACINA_CORS_ALLOWED_ORIGINS on the server to
the URL users load the UI from (REQ-009). The serverUrl value here is the
address the *browser* uses to reach the server (its ingress/public URL),
not the in-cluster Service DNS.
